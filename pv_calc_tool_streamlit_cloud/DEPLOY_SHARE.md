# 光伏项目自动测算工具公网分享说明

当前桌面快捷方式打开的是本机地址，例如 `http://localhost:8501`。这个地址只能在你自己的电脑访问，微信发给别人后，对方无法打开。

要在微信里分享，需要把工具部署到一台公网服务器，并使用公网 IP 或域名访问。

## 推荐方案

### 方案一：云服务器部署

适合内部长期使用，推荐腾讯云、阿里云、华为云等。

服务器建议配置：

- 系统：Ubuntu 22.04 或 Debian 12
- CPU：2 核及以上
- 内存：4 GB 及以上
- 带宽：2 Mbps 及以上
- 开放端口：`8501`，正式使用建议通过 Nginx 绑定域名和 HTTPS

部署步骤：

```bash
cd pv_calc_tool
docker build -t pv-calc-tool .
docker run -d --name pv-calc-tool -p 8501:8501 --restart=always pv-calc-tool
```

访问地址：

```text
http://服务器公网IP:8501
```

然后把这个地址发到微信即可。

### 方案二：公司内网服务器部署

适合只给公司内部人员使用。部署方法和云服务器一样，但只能在公司网络或 VPN 内访问。

### 方案三：Streamlit Community Cloud

适合演示，但需要把代码放到 GitHub。由于本工具涉及项目测算、电费单、屋顶截图等内部数据，不建议直接放公开仓库。

## 正式分享建议

1. 使用域名，例如 `https://pv.broad-sun.com`。
2. 使用 HTTPS，避免微信提示不安全。
3. 加登录密码或内网访问控制，避免测算数据被外部人员看到。
4. 上传的电费单、屋顶截图可能包含企业敏感信息，正式部署前建议增加文件清理或账号权限。

## 本地测试命令

```bash
cd pv_calc_tool
streamlit run app.py
```

本地地址仍然是：

```text
http://localhost:8501
```

这个地址只能本机访问，不能直接微信分享。
