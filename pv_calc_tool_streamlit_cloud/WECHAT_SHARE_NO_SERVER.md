# 不买服务器，微信分享使用方法

这个工具是 Streamlit/Python 网页应用，必须有一台机器在运行它。  
如果不买服务器，最简单的方法是：用你的电脑运行工具，再通过“内网穿透”生成一个公网链接。

## 推荐方式：内网穿透

可选工具：

- ngrok：国际通用，生成 `https://xxxx.ngrok-free.app` 这类链接。
- cpolar：国内访问通常更方便，适合临时演示和微信分享。

## 使用方式

### 方式一：ngrok

1. 安装 ngrok 并登录账号。
2. 双击运行：

```text
start_share_ngrok.cmd
```

3. 复制窗口里 `Forwarding` 后面的 `https://...` 链接。
4. 把链接发到微信。

### 方式二：cpolar

1. 安装 cpolar 并登录账号。
2. 双击运行：

```text
start_share_cpolar.cmd
```

3. 复制窗口里生成的 `https://...` 链接。
4. 把链接发到微信。

## 注意事项

1. 你的电脑必须一直开着。
2. 光伏测算工具和穿透窗口都不能关闭。
3. 免费链接可能会变化，每次重新打开可能要重新发链接。
4. 如果上传了电费单、屋顶截图等敏感资料，不建议把链接发给无关人员。
5. 若要长期稳定使用，还是建议部署到云服务器或公司内网服务器，并加账号密码。

## 为什么不能直接发桌面快捷方式

桌面快捷方式打开的是：

```text
http://localhost:8501
```

`localhost` 只代表你自己的电脑。别人手机打开这个地址时，会访问他自己的手机，而不是你的电脑，所以打不开。
